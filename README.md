# 🚨 VulnAlert Bot

Real-time security monitoring bot that detects vulnerability-related issues in GitHub repositories and instantly alerts security teams via Slack.

## ⚡ Features
- Real-time GitHub issue monitoring
- Detects 15+ security keywords
- Instant Slack alerts with formatted messages
- Serverless architecture — zero maintenance
- Closed-loop confirmation back to GitHub

## 🛠️ Technologies
n8n | GitHub Webhooks | Slack API | REST APIs | JSON
